# missing-osm

This project is a map interface that visualizes missing OSM buildings.

## Project Setup


### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

```sh
npm run lint
```
